# NotificationsBotAI

Telegram-бот для создания, управления и планирования уведомлений и напоминаний с поддержкой повторяющихся событий, локализации (английский/русский) и веб-API для истории операций и очистки данных.

## Возможности
Создание разовых и повторяющихся уведомлений (ежедневные, еженедельные, ежемесячные, ежегодные)

Быстрые таймеры (15 минут, 30 минут, 1 час) и ввод произвольной даты/времени (поддержка естественного языка)

Напоминания перед событием (за 15 минут, 1 час, 1 день)

Редактирование и удаление уведомлений

Просмотр истории операций

Настройка часового пояса и языка для каждого пользователя

Приветственные и справочные сообщения для новых пользователей

Автоматическая очистка старых уведомлений и неактивных пользователей

Корректная обработка заблокированных пользователей

Поддержка локализации на английском и русском языках


## Android-приложение Notifications

Мы также планируем разработать полноценное Android-приложение, которое расширит возможности Telegram-бота и обеспечит более удобный пользовательский интерфейс.

## Цель приложения

Создать удобный, быстрый и интуитивно понятный инструмент для управления напоминаниями и уведомлениями с возможностью синхронизации с Telegram-ботом и веб-API.
